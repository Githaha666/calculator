# Calculator

Just a simple calculator program by a Chinese pupil.

## Usage

You can call ```calculator.calculate()``` function to calculate the result of an expression.

For example:

```python
import calculator
result = calculator.calculate('2 + 3 * 4')
print(result)  # Output: 14
```

Unit tests: test.py
You can call:

```python
import calculator.test
calculator.test.tests()
```

to run the unit tests.
